Question-1: What are the different ways to select an element in the DOM?

Ans: In the DOM there are several ways to select elements using JavaScript.
1.getElementById("id");
2.getElementsByClassName("class-name");
3.getElementsByTagName("div");
4.getElementsByName();
5.querySelector(".Class-name");
6.document.querySelectorAll(".Class-name");

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans:1. By using innerHTML We can get or set html content inside an element.
2.By using innerText we can get or set text inside an element.but ignores hidden elements.
3.By using textContent we can get or set all text content,including hidden elements.

Question-3: What is event delegation in the DOM?

Ans: Event Delegation is a event bubbling technique in javascript.by This we can add a single addEventListener to the parent element and this will handle the multiple child element events.

Question-4: What is event bubbling in the DOM?

Ans: Bubbling in the DOM is an event propagation method.In This an event starts from the target element and bubbles up through its parent elements until it reaches the document root.
